# Premier School Exhibition Landing Page

A modern, fully responsive landing page developed for a school exhibition event. This project showcases a premium UI design with smooth layouts, responsive components, and an engaging user experience.

---

## 🚀 Live Demo

🔗 Netlify Link: https://premier-school-exhibition-website.netlify.app/

---

## ✨ Features

- Fully responsive design (Desktop, Tablet, and Mobile)
- Modern hero section with gradient background
- Animated vertical image gallery
- Mobile-friendly horizontal image slider
- Interactive buttons with hover effects
- School logo showcase section
- Responsive card layouts
- Exhibition information section
- Appointment call-to-action section
- Modern footer with contact information and social icons
- Smooth scrolling sections

---

## 📱 Responsive Breakpoints

| Device | Screen Size |
|---------|------------|
| Desktop | 1600px and above |
| Laptop | 1024px - 1600px |
| Tablet | Below 1024px |
| Mobile | Below 768px |
| Small Mobile | Below 480px |
